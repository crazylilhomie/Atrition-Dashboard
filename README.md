Streamlit EA Attrition Dashboard

Files in this package:
- app.py : Main Streamlit application
- README.md : This file

How to use:
1. Upload this repo to GitHub (single files, no folders required).
2. In Streamlit Cloud, connect to the GitHub repo and deploy.
3. Ensure your data `EA.csv` is available at `/mnt/data/EA.csv` for local default; when deployed, use the upload tab to provide data.

Dependencies (install via pip in Streamlit Cloud):
streamlit
pandas
numpy
scikit-learn
plotly
matplotlib
